# Dev Setup

Automated setup tool for programming languages,
compilers and development environments.

## Features

- Git Installer
- VS Code Installer
- GCC Installer
- Python Installer (WIP)

## Roadmap

- Java Support
- Node.js Support
- Flutter Support
- Linux Support
- MacOS Support


## Installation

```bash
git clone https://github.com/TaH00R/dev_setup.git
cd dev_setup
python main.py
```
